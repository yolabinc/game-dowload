Description

Fortnite is a well-accepted online video game developed& published by Epic Games. The first early access version was released on July 25, 2017. The early release included two modes: Save the World and Battle Royale.

Save the World is a cooperative mode where players can team up to fight AI-controlled enemies, build fortifications, and gather resources. It was set in a post-apocalyptic world where the majority of humanity has been wiped out by a mysterious storm.

Battle Royale is a competitive mode where players fight against each other in a last-man-standing deathmatch. The game takes place on a large island map, and players can scavenge for weapons and resources. The map gradually shrinks over time, forcing players to fight until there is only one winner.

Later in 2018, Fortnite released a third game mode called Creative. This mode allows players to create their custom island maps and play on them with friends. It has no limitations, and players can use it to create any type of game they want.

Fortnite has been availed for several platforms, including Microsoft Windows, macOS, iOS, Android, PlayStation 4, and Xbox One. The game has been well-received by critics and is one of the most popular games in the world.
